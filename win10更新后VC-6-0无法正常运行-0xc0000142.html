Win10更新后VC++6.0无法正常运行(0xc0000142)

发表于 2016-10-09 | 分类于 Tech | |

今天一同学原来能正常运行VC++6.0，更新win10后，打开VC++6.0，一直出现应用程序无法正常运行（0xc0000142），通过卸载重装VC，使用兼容XP，修改文件名字还是没有解决问题，原来是win10不兼容中文版的VC，最后用下面的方法解决了。

  1. 下载并解压我分享的英文版MSDEV.EXE，链接：http://pan.baidu.com/s/1nvfLGaX 密码：lpqw
  2. 找到自己安装VC的路径，默认安装在C盘C:\Program Files (x86)\Microsoft Visual Studio\Common\MSDev98\Bin 这个目录下，笔者安装在D盘D:\learn\Microsoft Visual Studio\Common\MSDev98\Bin，沿着Microsoft Visual Studio\Common\MSDev98\Bin这个路径就好。
  3. 将第一步解压后的MSDEV.EXE文件复制替换到第二步的目录下。
  4. 复制粘贴过后，右键点击MSDEV.EXE发送到桌面快捷方式即可。

【注】一定要按照步骤和复制粘贴到正确的文件目录下。操作不成功的检查下是否出错。

希望大家都能成功解决这个问题，好尽情的编写程序。嘿嘿~

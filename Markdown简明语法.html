Markdown简明语法总结

发表于 2016-12-05 | 分类于 Tech | |

最近两周一直忙着考试，今天终于有时间来继续探索我的个人博客了。因为每次写博客都是用 Markdown 写的，其中很多技巧还没有了解，趁着今天赶紧学习下。方便我以后使用。当你用了 Markdown 后，你会发现其实 Markdown 语言很好用。现在很多网站，软件也都支持 Markdown 的使用，比如，有道云笔记，简书等。

大家可以看下这篇《Markdown写作简谈》，笔者也很推荐大家使用Markdown语言。以下是笔者总结的Markdown简单常用的语法。

基本符号

  • *,-,+ 3个符号效果一样，被称为Markdown符号
  • 空白行表示另起一个段落，也就是回车两次

标题

写文章时，标题不可或缺，在 Markdown 中，在文本前面加上 # 即可表现标题形式，你还可以增加二级标题、三级标题、四级标题、五级标题和六级标题，总共六级，只需要不断增加 # 即可，标题字号会随着标题级数逐渐降低。

【注】 # 和一级标题之间最好保留一个字符的空格，这是 Markdown 最标准的写法。

引用

文章引用很常见，在 Markdown 中，我们只需要在引用的文字前面加上 “>” 和一个空格，这样就可以表示引用了。

列表

列表在 Markdown 文件中是主要表示方式。在 Markdown 中，你只需要在文字前面加上 “-“加上一个空格就可以了，如果想要有序列表，可以用数字 + 英文点 + 空格来表示。

图片和链接

写文章有时候会想配一个有当时意境的图片，在 Markdown 中我们需要这样表示：![图片名称](图片链接地址)。

有时候会需要一个超链接，添加链接和图片方式比较相似，只是前面不用加 ！ ，即[显示文本内容](链接地址)。

此外，还可以以索引方式把url都列在文章的最后，例如这样：

![图片名称](1)

[显示文本内容](2)

[1]:图片链接地址

[2]:链接地址

粗体和斜体

Markdown 的粗体和斜体也非常简单，用两个 包含一段文本就是粗体的语法，用一个 包含一段文本就是斜体的语法。

比如：*好好学习* **天天向上**

最终显示：好好学习 天天向上

代码引用

作为一个程序员，文章中代码的引用是很常见的，在 Markdown 中，如果引用的语句只有一段，不分行，可以用符号 “`” 将置于语句前后。

如果引用的语句为多行，可以分别将3个符号 “`” 连起来置于这段代码的首行和末行。

表格

关于表格的代码如下：

Name Class Score
Meng 2 100
Lei 2 100

结语

以上是我总结的关于 Markdown 常用的语法，如果还想了解更多有关语法，请参考《Markdown语法说明》。
